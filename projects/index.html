Projects

Updated Nov 16, 2023

Here's a comprehensive list of projects that I've completed or am presently maintaining from my undergraduate education onwards.

  • ~ Web Crawler and Search Engine

    Built web crawler and naïve search engine with Java, HTML/CSS, and JS. Indexes webs of size 100k+ and performs efficient queries with full logical operator support. Indexes user's tokenized queries to provide trend visualization with D3.js library and generative search suggestions. Implemented Python web-generation script to create large, controllable webs for testing. Created for Dr. Lin's CS 314H course in Fall 2023.

  • ~ Bounding Volume Hierarchy

    Java implementation of the spatial BVH acceleration data structure. Supports insertions/removals and maintains AVL balance property for common-case logarithmic performance. Created for Dr. Lin's CS 314H course in Fall 2023.

  • ~ Genetic JBoggle

    Wrote game engine and implementation for the word game Boggle in Java. Includes efficient recursive algorithms to find all words present on a board utilizing a Trie dictionary structure. Wrote a genetic optimization algorithm to find the highest-scoring Boggle board. Created for Dr. Lin's CS 314H course in Fall 2023.

  • ~ Genetic JTetris

    Game logic and mechanics for a Tetris engine built in Java; created a genetic algorithm from scratch to automate the process of playing. After 50 generations of training with a population of 1,000, it consistently survives 500,000 tetrominoes. Created for Dr. Lin's CS 314H course in Fall 2023.

  • ~ Critterlang Interpreter

    The Critterlang Interpreter is an interpreter designed and built in Java for a low-level instruction language used to create digital critters for Turing's annual CritterFest programming competition. Created for Dr. Lin's CS 314H course in Fall 2023.

  • ~ Random Writing

    Random Writing is a program in Java to produce file output writing similar to an input text by storing token data and semi-randomly generating new expressions via Markov chains. Created for Dr. Lin's CS 314H course in Fall 2023.

  • ~ Image Manipulation

    Image Manipulation is a collection of logic and manipulation algorithms for a Java Image Processor (JIP) application that applies transformations to images' corresponding pixel matrices; implemented transformations include color filters, reflections, rotations, resizers, and convolution effects. Created for Dr. Lin's CS 314H course in Fall 2023.

  • ~ DLSort

    DLSort is a simple, Python-based file sorting script. I built it to automate the file organization (especially for my classwork) on my machine running macOS—because why not have a computer do the work for you?
