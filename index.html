Projects

    Bounding Volume Hierarchy

    Java implementation of the spatial BVH acceleration data structure. Supports insertions/removals and maintains AVL balance property for common-case logarithmic performance.

    Genetic JBoggle

    Game engine and implementation for the word game Boggle in Java. Includes efficient recursive algorithms to find all words present on a board utilizing a Trie dictionary structure. Wrote a genetic optimization algorithm to find the highest-scoring Boggle board.

    Genetic JTetris

    Game logic and mechanics for a Tetris engine built in Java; created a genetic algorithm from scratch to automate the process of playing.

    Random Writing

    Java program to produce file output writing similar to an input text by storing token data and semi-randomly generating new expressions via Markov chains.
