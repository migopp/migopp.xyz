All Horses Are The Same Color

August 20, 2023

It's true‚Äîpreposterous, but true. All horses are the same color.

Argument

For a population of 1 horse, we can say for certain that it has the same color as itself. Assume that any group of n horses all have the same color, where n is an arbitrary positive integer 1. Now, consider a population of n + 1 horses.

We can split the n + 1 horses into two groups, the first n horses, and the last n horses‚Äîas shown below:

In either case, we know that each set contains all of the same color because of our assumption 2. Since the first horse is the same color as the middle horses, and the last horse is the same color as the middle horses, the first and last horses must be the same color‚Äîand, of course, must share the same color as the middle horses. So, all n + 1 horses are the same color. So, the existence of a group of n horses of the same color implies that a group of n + 1 horses will have the same color. So, any amount of horses will have the same color. All horses have the same color, if you will. QED, proof done‚Äîshown beyond a reasonable doubt.

Right?

Fallacy

Of course not. I lied. We know that not all horses are the same color. There are white horses, brown horses, grey horses, and pink 3 horses: among many other kinds.

So, how is our argument flawed?

The underlying method of proving this bologna assertion is actually a valid way of proving properties for all objects with assumed shared characteristics. It's called induction. The idea is that, if you can prove a property holds for one object (called verifying the base case), and that the veracity holds while adding another object (the inductive step), you can prove that statement for all of the objects‚Äîbecause, you could continue repeating the inductive step forever.

The assumption is also not flawed. This is a crucial part of induction, but if you have proven the base case to be truthful, then the arbitrary n could be chosen as that base case. In that situation, you are simply proving that the base case implies the following one. It's nothing extraordinary, though it seems to be on the outset.

In fact, our proof works for almost all n. All, but a single value. For the value n + 1 = 2, (or, more simply n = 1), the inductive step does not work because there are no middle horses to compare the starting and ending horses to. In this case, not considering everything on the small scale hindered our ability to recognize dissonance on the large scale. In fact, the truly vital case was not the case where there was one horse, but the case where there became multiple of them. That is truly the shift that deserves special attention.

It's easy (in the way that I've presented this proof) to be ‚Äúmisled by the ellipsis‚Äù as Albert Meyer put it. We assume that the small scales to the large, so we need not consider it. We assumed that proving it for all horses required of thinking large, thinking of the case where there are a lot of them and they ought to differ from each other somehow. We assumed a lot of horses were behind the three dots; however, as we've seen, the vital case is‚Äîperhaps perplexingly‚Äîthe inverse.

So, as you've seen, with a little trick, something absurd can seem true. Such is the case with many fallacious arguments, so being able to break everything down and understand each component is of the utmost importance.
